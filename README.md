# لوحة تحكم مشروع راف 25 🏢

لوحة تحكم شاملة ومتطورة لإدارة موقع مشروع راف 25 السكني في جدة. تتيح لك التحكم الكامل في جميع محتويات الموقع بسهولة وبساطة.

## ✨ المميزات الرئيسية

### 📊 لوحة التحكم الرئيسية
- إحصائيات شاملة للموقع
- إجراءات سريعة للوصول للوظائف المهمة
- عرض النشاط الأخير
- تصميم عصري ومتجاوب

### 🏠 إدارة الشقق
- تعديل تفاصيل جميع نماذج الشقق
- إدارة الأسعار والمساحات
- تعديل عدد الغرف ودورات المياه
- إضافة/حذف مكونات الشقق
- تحديد الشقق الأكثر طلباً

### 💰 إدارة الأسعار
- تعديل الأسعار الأساسية
- إدارة العروض الخاصة
- خطط الدفع المتعددة
- حاسبة الأسعار التفاعلية
- إدارة العملة وفترات التقسيط

### 🖼️ إدارة الصور
- رفع وتعديل صور المشروع
- تصنيف الصور (بانر، شقق، معرض)
- ترتيب الصور حسب الأولوية
- معاينة مباشرة للصور
- تحسين SEO للصور

### 📞 معلومات التواصل
- إدارة أرقام الهاتف والواتساب
- تحديث البريد الإلكتروني
- إدارة العنوان وساعات العمل
- رسائل الواتساب الافتراضية
- أرقام الطوارئ والمبيعات

### 📍 الموقع والخريطة
- تحديث إحداثيات المشروع
- إدارة خريطة جوجل التفاعلية
- إضافة الأماكن القريبة
- مميزات الموقع الإستراتيجي
- مشاركة الموقع مباشرة

### 🌐 وسائل التواصل الاجتماعي
- إدارة جميع الحسابات الاجتماعية
- تفعيل/إلغاء تفعيل الحسابات
- تحديث الروابط وأسماء المستخدمين
- فتح الحسابات مباشرة
- نصائح لإدارة المحتوى

### ⚙️ إعدادات الموقع
- تعديل العناوين والوصف
- إدارة المحتوى العام
- إعدادات الأقسام المختلفة
- معلومات الترخيص والرقم الموحد
- معاينة الإعدادات

### 📋 طلبات الحجز
- عرض جميع طلبات الحجز
- تصفية حسب الحالة
- تحديث حالة الطلبات
- التواصل المباشر مع العملاء
- إحصائيات مفصلة

## 🚀 التقنيات المستخدمة

- **Frontend:** Next.js 15, React 19, TypeScript
- **Styling:** Tailwind CSS v4
- **UI Components:** Radix UI
- **Icons:** Lucide React
- **State Management:** React Hooks
- **Design:** RTL Support, Arabic Typography

## 📱 التصميم والمظهر

- **تصميم متجاوب:** يعمل على جميع الأجهزة
- **دعم كامل للعربية:** RTL والخطوط العربية
- **واجهة عصرية:** تصميم نظيف وسهل الاستخدام
- **ألوان متناسقة:** نظام ألوان موحد ومتسق
- **تجربة مستخدم ممتازة:** انتقالات سلسة وتفاعل سريع

## 🛠️ التثبيت والتشغيل

### المتطلبات
- Node.js 18+
- npm أو pnpm

### خطوات التثبيت

1. **استنساخ المشروع**
```bash
git clone [repository-url]
cd jeddah-residential-landing
```

2. **تثبيت التبعيات**
```bash
npm install
# أو
pnpm install
```

3. **تشغيل المشروع**
```bash
npm run dev
# أو
pnpm dev
```

4. **الوصول للوحة التحكم**
```
http://localhost:3000/admin
```

## 🎯 الاستخدام

### للوصول للوحة التحكم:
1. افتح المتصفح واذهب إلى `/admin`
2. ستجد لوحة التحكم الرئيسية مع الإحصائيات
3. استخدم الشريط الجانبي للتنقل بين الأقسام

### لتعديل المحتوى:
1. اختر القسم المطلوب من القائمة
2. اضغط على زر "تعديل"
3. قم بالتغييرات المطلوبة
4. اضغط "حفظ" لتطبيق التغييرات

### لإدارة الصور:
1. اذهب إلى قسم "الصور"
2. اختر "رفع صورة جديدة"
3. حدد الصورة واكتب الوصف
4. اختر الفئة المناسبة
